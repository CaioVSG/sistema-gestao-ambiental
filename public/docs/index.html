Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost:8003

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer your-token".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Endpoints

Visitas

requires authentication

Consulta todas as visitas do usuário autenticado.

Example request:
curl --request GET \
    --get "http://localhost:8003/api/visitas" \
    --header "Authorization: Bearer Ed63bgP6f1DaVk54ae8hvZc" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8003/api/visitas"
);

const headers = {
    "Authorization": "Bearer Ed63bgP6f1DaVk54ae8hvZc",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Autenticação

Cria um token de autenticação para o usuário.

Example request:
curl --request POST \
    "http://localhost:8003/api/users/auth" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"cupiditate\",
    \"password\": \"id\",
    \"device_name\": \"aut\"
}"

const url = new URL(
    "http://localhost:8003/api/users/auth"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "cupiditate",
    "password": "id",
    "device_name": "aut"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
